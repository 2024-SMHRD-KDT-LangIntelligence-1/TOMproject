# 📎추천 알고리즘을 활용한 나만의 가계부 웹사이트 LuckyPiggy (팀명: TOM)


## 👀 서비스 소개
* 서비스명 : 추천 알고리즘을 활용한 나만의 가계부 웹사이트
* 서비스설명
       1. 사용자 기반 추천 알고리즘을 활용한 카드 추천 서비스
       2. 대시보드를 통한 재정 관리 및 카드 실적 관리 서비스 
<br>

## 📅 프로젝트 기간
2024.12.23 ~ 2025.01.08
<br>

## ⭐ 주요 기능
* 사용자 기반 카드 추천 
* 캘린더(월간 소비 내역)
* 일일 소비 내역
* 검색기능
* 입출금 그래프와 소비차트
<br>
